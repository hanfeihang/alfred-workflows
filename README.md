# alfred-workflows

Some useful alfred workflows


# List

- Timestamp transfer from/to Date
    
    - Usage
        
        use alfred keyword `tt`
        ```
        tt 2019-06-26 12:00:00 -> 1561521600000
        tt 1561521600000       -> 2019-06-26 12:00:00
        tt                     -> 1561521600000 / 2019-06-26 12:00:00
        ```
    - Dependency
        
        NodeJs
